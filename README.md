# MultiConverterBoard

Multi Converter Board V1.1 to realice tests in electronics lab and understand the operation of the next schemes: 


- Buck asynchronous 

- Boost asynchronous 

- Buck-Boost asynchronous 

- Sepic asynchronous
 
- Zeta asynchronous 

- Flyback asynchronous 

- Buck synchronous 

- Boost synchronous 

- Buck ZCS 

- Buck ZVS 

- Boost ZCS 

- Boost ZVS
